## 📂 Requirements
Make sure you have the following installed on your computer/device:
- ✅ Visual Studio Code ([download here](https://code.visualstudio.com/)).
- ✅ Node JS ([download here](https://nodejs.org/en)).
- ✅ Git ([download here](https://git-scm.com/downloads)).

## 👉 Get Started 
Run Project Locally
<details>
<summary><code>1. Clone the repository (Source Code)</code></summary>

  >
  > In your Desktop, open a command prompt of your choice (git bash, cmd, or any)
  > ```bash
  > git clone https://github.com/KJLEscoto/remindr.git
  > ```
  > Go to main directory
  > ```bash
  > cd remindr
  > ```
  > Open folder to VS Code
  > ```bash
  > code .
  > ```
</details>

<details>
<summary><code>2. Install Dependencies for Nuxt 4</code></summary>
  
  > Install modules
  > ```bash
  > npm install
  > ```
  > Fix compatibilities
  > ```bash
  > npm audit fix
  >```
  > Run the server
  > ```bash
  > npm run dev
  > ```
</details>
